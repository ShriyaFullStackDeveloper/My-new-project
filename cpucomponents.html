CPU COMPONENTS
Sr. no. Components Image Description
1 ALU (Arithmetic Logic Unit) Performs arithmetic and logical operations.
2 Control Unit (CU) Directs operations in the CPU by interpreting and executing instructions.
3 Registers Small, high-speed storage locations that hold data and instructions temporarily during processing.
4 Cache Memory A small, fast memory located inside the CPU used to store frequently accessed data for quicker access.
5 Clock Generates a consistent timing signal to synchronize all operations within the CPU.
6 Buses Communication pathways that transfer data between components inside the CPU.
7 Instruction Decoder Decodes instructions from the program and translates them into signals that control other parts of the CPU.
8 Execution Unit (EU) Part of the CPU that carries out the operations defined by the instructions.
9 Floating Point Unit (FPU) Specialized unit for handling floating-point calculations, like division and multiplication of non-integer numbers.
10 Cache Controller Manages the data flow between the CPU cache and other system components like RAM.
11 Branch Predictor Helps to predict the direction of branches in the code to improve the flow of instructions.
12 Translation Lookaside Buffer (TLB) A cache that stores recent translations of virtual memory addresses to physical memory addresses.
13 Data Bus Carries data between the CPU and other system components.
14 Address Bus Carries memory addresses from the CPU to other components such as RAM or storage.
15 Power Management Unit (PMU) Controls the power usage and manages the CPU's energy efficiency.
